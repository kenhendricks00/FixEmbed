# FixEmbed Premium store listing

Power up your server with more control over how social links look and behave while supporting FixEmbed's continued development.

## Included Premium features

- 🎨 **Custom Social Card Colors** — Match every supported Components V2 social card to your server's style.
- 🎛️ **Card Style Controls** — Show or hide engagement stats and hashtags, and choose full or compact captions.
- 🛡️ **Advanced Exclusions** — Leave links from selected members or roles untouched.
- 📈 **Private 30-Day Analytics** — See aggregate platform usage and fallback reliability. FixEmbed does not store message text, links, or member identities for analytics.
- 🏷️ **Custom Footer Branding** — Use your server name and an optional server emoji while retaining subtle FixEmbed attribution.
- 🤖 **Bot Link Processing** — Fix links posted by other bots.
- ✨ **Cleaner Automatic Posts** — Remove the standard `Sent by` label from automatic replacements.
- 💎 **Supporters Role** — Receive the Supporters role in the FixEmbed Support Server while Premium is active.

Core embed reliability, media playback, GIFs, carousels, all-platform translation settings, translations requested in a link, and media-quality controls remain available to everyone.
