# Triage labels

| Skill role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer evaluation is required |
| `needs-info` | `needs-info` | More information is required from the reporter |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for implementation |
| `ready-for-human` | `ready-for-human` | Requires human action or access |
| `wontfix` | `wontfix` | Will not be actioned |

When a skill names a triage role, use the corresponding GitHub label in this table.
