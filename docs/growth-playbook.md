# FixEmbed Growth Playbook

This playbook turns FixEmbed's product quality into repeatable,
privacy-respecting acquisition. Normal social-card footers stay quiet. Growth
comes from useful product surfaces, clear demonstrations, release notes, and
real user proof.

## Core funnel

1. A Discord user sees a rich FixEmbed card or a focused platform page.
2. **Install to My Account** is the primary action because it works in DMs,
   group chats, and servers without requiring administrator access.
3. **Add to Server** remains visible for automatic conversion and settings.
4. The website records only the fixed placement label and install context when
   it redirects to Discord. A redirect is a directional click, not a confirmed
   install.

Review these weekly:

- account-install and server-install redirect clicks by fixed placement;
- active guilds with at least five successful conversions in seven days;
- first successful conversion after install;
- four-week active-guild retention;
- rich-card, link-rescue, and complete-failure rates already exposed by the
  reliability system.

Do not join these counts to Discord IDs, member data, message content, or social
media URLs.

## 20-second demonstration storyboard

Use a real test-server recording with permission-safe public posts. Crop out
unrelated member names and channels.

| Time | Shot | On-screen copy |
|---|---|---|
| 0-3s | Paste an ordinary X, Instagram, or Reddit URL | "Discord deserves the whole post." |
| 3-8s | FixEmbed replaces it with the Components V2 card | "Media, author, stats, and original time." |
| 8-13s | Play a video/GIF and show a carousel or quote | "Playable. Complete. In Discord." |
| 13-17s | Run `/fix` in a DM with the personal install | "Works anywhere you use Discord." |
| 17-20s | Show the two install choices | "Install to your account — or add to a server." |

Export a silent-captioned vertical version for Discord/social posts and a
16:9 version for the website/App Directory. Do not publish a mock recording as
a real product demonstration.

## Release announcement template

> FixEmbed now makes personal installation the fastest way to fix social links
> anywhere in Discord. Install it to your account for `/fix` and the message
> command in DMs, group chats, and servers—or add it to a server for automatic
> conversion. The new X, Instagram, and Reddit pages show exactly what each
> card preserves. [Install FixEmbed](https://fixembed.app/)

For platform-specific communities, link the matching `/twitter`, `/instagram`,
or `/reddit` page and show one real before/after example. Never mass-DM users or
post in communities that prohibit bot promotion.

## Directory copy

**Short description**

> Fix broken social previews with modern, playable Discord cards for X,
> Instagram, Reddit, Threads, Pixiv, Bluesky, Bilibili, YouTube community posts,
> and Pinterest.

**Primary promise**

> Install FixEmbed to your account and use `/fix` or Apps → Fix Embed anywhere.
> Add it to a server when you want automatic conversion and server controls.

## Testimonial intake

Only publish a testimonial when all four fields are recorded:

1. the exact quote supplied by the user;
2. the public display name and server/community attribution they approved;
3. explicit permission to publish it on FixEmbed-owned surfaces;
4. a contact or support-thread reference for later removal requests.

Lightly correct spelling only after the user approves the final quote. Never
invent adoption numbers, customer names, or endorsements. Remove a quote
promptly when its author asks.

## Experiment rules

- Change one major promise or CTA order at a time.
- Run long enough to cover at least two comparable weekly cycles.
- Prefer activation and retained usage over raw clicks.
- Keep the social-card footer free of Premium pitches and install CTAs.
- Do not gate reliable embeds, playback, GIFs, carousels, or platform fixes
  behind votes or Premium.
